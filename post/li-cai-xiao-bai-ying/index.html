理财小白营

2019-09-01

微信上报了个9元的课程，周末抽空补了下进度，还是尽快去补完进度，每天坚持了解一些。

入门阅读

1.《小狗钱钱》

  • 1.1 想变的有钱，这3个建议可以帮你
    • 1.1.1 缺钱的时候，钱就格外重要
    • 1.1.2 不想一辈子就穷下去，现在就要学会理财
    • 1.1.3 人的精力有限，删除干扰选项，才更有机会实现愿望
    • 1.1.4 永远不要试试看，只有“做”或“不做”
    • 总结
      • 写下Top3愿望
      • 制作梦想相册——激励和推动自己
      • 准备梦想储蓄罐
  • 1.2 赚钱没你想的那么难
    • 1.2.1 写“成功日记”
      • 任何小事都可以，给自己加油、自信
    • 1.2.2 商人赚钱建议
      • 为别人解决问题
      • 把精力放在你知道的事情和能够做的事情上
    • 1.2.3 觉得钱难赚的人
      • 没有认真思考
        • 赚钱你要花费多少时间？
      • 不愿面对困难
        • 找借口，放弃
    • 总结
      • 写“成功日记”，增加自信
      • 别人需要什么？
      • 我能做什么？
  • 1.3 99%的人不成功，都是因为它
    • 1.3.1 遇到困难的时候，越要坚持自己的想法
    • 1.3.2 要每天不间断地去做对你未来意义重大的事
      • 21天理论——坚持21天就会变成习惯
    • 1.3.3 当决定做一件事的时候，就要立即去做
      • 否则很可能永远不会再去做
    • 总结
      • 坚持最重要
  • 1.4 不要杀死你的鹅
    • 1.4.1 不要杀死你的鹅
      • 不要花光本金，否则永远不会有收益
    • 总结
      • 负债的人应该扔掉所有信用卡
        • 使用信用卡，总比使用现金花的多
      • 尽可能少的偿还贷款
        • 如果是信用卡债还是建议先还债
        • 还要考虑通货膨胀，以后钱会缩水
      • 对于消费贷款，应该将不用于生活的钱存起来一半，另一半用于还债
        • 存钱，保证储蓄(本金)
      • 把想要的东西分为“必要、需要、想要”
        • 控制消费
  • 1.5 投资，遵循这3个原则就足够了
    • 1.5.1 现金的重要性，应对突发情况
      • 投资前，做好资产配置
    • 1.5.2 金钱能否带来幸福，取决于是否懂得怎样使用它
    • 1.5.3 穷人思维
      • 拿到钱买买买，吃喝玩乐，奢侈享受，忽略了投资
    • 1.5.4 富人思维
      • 消费一些钱作为奖励，将80%甚至更多的钱投资，继续赚取更多的钱
    • 总结
      • 投资要安全
        • 本金不能丧失
      • 要有收益
        • 先评估风险，再考虑收益多少
      • 充分了解投资对象
        • “不懂的就不要碰”

2.【晚间分享】财富的载体——时间&量化思维

  • 2.1 大多数人在工作和生活中，更多地注意金钱，而很少注意“时间”
    • 有钱人会感觉时间比金钱重要，愿意用钱买时间
    • 没钱的人更在乎钱，不在乎时间，愿意用时间做一些没什么价值的事情
  • 2.2 穷人思维
    • 因为没钱，所以缓缓；等有钱了，怎样怎样
    • 为免费、特价等，付出过多的资源和时间
      • 可能浪费过多的时间和注意力
  • 2.3 理财思维
    • 重视时间成本
  • 2.4 不理财不等于没风险，通货膨胀其实就是最大的风险
  • 2.5 “怕吃苦，吃苦一辈子，不怕苦，吃苦一阵子”
  • 总结
    • 每个人的时间都有一定的金钱价值
      • 用凡事不要凭感觉，用量化思维计算，选择更有价值的事做
    • 金钱是有时间价值的，现在的钱比未来更值钱
      • 提高投资能力，让钱更值钱
    • 学习是有时间成本的
      • 学的越晚，消耗越大
  • 拓展知识
    • 投资的不可能三角
      • 投资理财目标中，收益性、安全性、流动性三者，最多只能占两个
        • 收益性、安全性、！流动性
          • 房子
        • 安全性、流动性、！收益性
          • 活期存款、货币基金等
        • 收益性、流动性、！安全性
          • 部分P2P
      • 通过投资资产的三种属性，判断投资品种是否靠谱
