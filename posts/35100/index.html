umi新建ant design pro项目使用js

都是不能在非空目录建新项目

最后是建立项目后，移动目录

# npm install -g yarn
# yarn create umi # 每次会用最新版本的，不合适

# 第一种
npm create umi@0.27.1 myapp

## 第二种
yarn global add create-umi@0.27.1
cd ~/.yarn/bin/
ln -s ../../.config/yarn/global/node_modules/.bin/create-umi create-umi@0.27.1

yarn create umi@0.27.1 myapp
