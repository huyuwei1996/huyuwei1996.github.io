jvm-sandbox-repeater

流派 技术实现 优点 缺点
流量转发 diffy http报文转发，可以兼容不同语言开发的应用 无法应对写操作，回放有可能造成数据污染
tcpcopy/goreplay 对被测系统几乎不造成干扰 无法对应用层的数据进行筛选和修改，细节配置复杂，改造成本高
无法应对写操作，回放有可能造成数据污染
java agent sandbox repeater 对于Java语言开发的系统有很高的扩展性
可以有效应对写操作		 无法兼容其他语言开发的应用
开源版本的功能较少，需要二次开发
